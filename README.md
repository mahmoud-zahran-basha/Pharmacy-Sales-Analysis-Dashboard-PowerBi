# Pharmacy Sales Analysis Dashboard 💊📊

### مشروع تحليل مبيعات سلسلة صيدليات باستخدام Power BI

## 📝 وصف المشروع (Project Description)
هذا المشروع يهدف إلى تحليل البيانات المالية والتشغيلية لسلسلة صيدليات تضم 6 فروع. يركز التحليل على حساب الأرباح الحقيقية بعد خصم التكاليف والخصومات، ومراقبة أداء المنتجات المختلفة.

## 🚀 الميزات الرئيسية (Key Features)
* **تحليل المبيعات:** حساب إجمالي المبيعات بناءً على الكميات وأسعار البيع المرتبطة.
* **أداء الفروع:** مقارنة المبيعات بين الفروع وتحديد الفروع الأكثر كفاءة.
* **تصنيف المنتجات:** مراقبة مبيعات الأدوية ومستحضرات التجميل ومنتجات العناية.

## 🛠️ الأدوات المستخدمة (Tools)
* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Data Modeling** (Star Schema)
